# 📚 Library Management System

## 🌟 Overview
The **Library Management System** is designed to efficiently manage a library’s operations, including cataloging books, tracking borrowed items, and managing member information. This system is built using **TypeScript** and **JavaScript**, ensuring a robust and scalable solution for library management.

## 🚀 Features
✅ **Book Cataloging**: Add, update, and delete books in the library’s catalog.  
✅ **Member Management**: Register new members, update details, and manage membership statuses.  
✅ **Borrowing System**: Track borrowed books, due dates, and return statuses.  
✅ **Search Functionality**: Quickly find books and members.  
✅ **Reporting**: Generate reports on borrowed books, overdue items, and member activities.

## 📥 Installation
Follow these steps to set up the project locally:

1️⃣ **Clone the repository:**  
   ```bash
   git clone https://github.com/Daudimungai/Library-Management-System.git
   ```
2️⃣ **Navigate to the project directory:**  
   ```bash
   cd Library-Management-System
   ```
3️⃣ **Install dependencies:**  
   ```bash
   npm install
   ```
4️⃣ **Run the application:**  
   ```bash
   npm start
   ```

---

## 🎯 Usage
After starting the application, access it via **[http://localhost:3000](http://localhost:3000)**. 

### 🔹 Main Functionalities:
- 📊 **Dashboard** – View an overview of the library’s status.
- 📖 **Books** – Manage the book catalog.
- 🧑‍🤝‍🧑 **Members** – Manage library members.
- 🔄 **Borrow/Return** – Track book lending and returns.
- 📜 **Reports** – Generate insights on library activities.
